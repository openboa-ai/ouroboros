# Core Beliefs

- AutoKairos is an installable trading app, not just a repo or CLI.
- The app should include a resident agent by default.
- The product goal is money-making automated trading.
- The runtime should be local-first but portable to server or VPS deployment.
- Research should stay broad and exploratory.
- Live execution should stay narrow, deterministic, and auditable.
- Candidate changes should improve a future version of the system, not directly edit the currently live path.
- The user remains the ultimate owner of risk and intervention controls.
- Durable markdown should accumulate and become easier to reuse over time.
