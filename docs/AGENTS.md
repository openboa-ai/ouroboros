# AGENTS.md

This subtree is the canonical markdown system for AutoKairos.

## Rules

- Keep durable product and architecture knowledge under `docs/`.
- Keep active work in `docs/exec-plans/active/` until it stabilizes.
- Do not duplicate the same concept across several branches of the docs tree.
- Link across layers instead of copying large blocks of text.

## Subtrees

- `design-docs/`
  Stable beliefs, runtime model, provider model, and skill-system rules
- `product-specs/`
  Product behavior and trading specifications
- `exec-plans/`
  Active and completed plans
- `references/`
  Source-backed external analysis
- `generated/`
  Generated repository artifacts
