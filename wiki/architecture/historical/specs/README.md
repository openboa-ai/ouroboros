# Historical Specs

This directory contains specs that were retained for context but removed from the active spec gate.

The active spec gate is [../../specs/README.md](../../specs/README.md). A spec in this directory is
not current implementation guidance by location alone.

Common reasons a spec lives here:

- it predates the `TraderSystemRuntime` / `RuntimePlacement` naming reset
- it assumes central workflow, state-machine, or scheduler-style orchestration
- it describes proactive-standing, rebuild, read-admission, coalescing, or projection families that
  are beyond the current MLP-01 active baseline
- it is a broader essay rather than a current implementation contract
