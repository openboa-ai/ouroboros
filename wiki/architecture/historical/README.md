# Historical Architecture Material

This directory preserves architecture material that is useful as background but is not part of the
current MLP-01 active baseline.

Rules:

- Active implementation guidance lives in [../README.md](../README.md) and the active spec gate.
- Historical material may contain older names, broader platform ideas, scheduler-centric models,
  proactive-standing families, or pre-reset runtime assumptions.
- Do not treat a historical page as current truth unless a current active page explicitly promotes it.
- Prefer linking to historical pages only from history, ADR, or migration context.
