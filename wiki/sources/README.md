# Sources

This directory is the source-first research base for autokairos.

Start with [library/index.md](library/index.md), then move to
[synthesis/index.md](synthesis/index.md), and only after that return to `wiki/architecture/`.

The current research pass tracks 13 primary sources and rewrites them as deep notes with explicit
scope, mechanism, cited-fact, and vocabulary matrices.

## Reading Order

1. [library/index.md](library/index.md)
2. the relevant per-source deep notes in `library/`
3. [synthesis/index.md](synthesis/index.md)
4. `wiki/architecture/`

Use it for:

- articles
- engineering essays
- research write-ups
- repository notes
- cross-source synthesis rooted in those notes

Rules:

- source notes are evidence files, not architecture pages
- keep one deep note per tracked source link in `library/`
- keep cross-source interpretation in `synthesis/`
- prefer immutable provenance once a source note is adopted into `library/`
- update architecture only after the source notes and synthesis are good enough to support it
- use `inbox/` for newly dropped material
- use `library/` for normalized and stable source notes
- use `synthesis/` for derived comparison pages
