# Library

Store normalized and stable per-source research notes here.

Each tracked source should have exactly one deep note in this directory. Those notes should follow
the shared schema documented in [index.md](index.md):

- `Source`
- `What This Source Is`
- `Core Thesis`
- `Key Mechanisms / Architecture`
- `Important Passages Or Facts`
- `Vocabulary And Mental Models`
- `Transferable Lessons`
- `Non-transferable Baggage`
- `Open Questions / Tensions`

In the current pass, notes are intentionally matrix-heavy. Expect explicit:

- research scope matrices
- mechanism mappings
- cited-facts tables
- vocabulary / primitive tables

Start with [index.md](index.md).
